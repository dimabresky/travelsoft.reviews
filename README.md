# travelsoft.reviews
Модуль "Оставьте отзыв" для BitrixFramework

Для установки модуля папку с файлами необходимо скопировать в /local/modules или клонировать
репозиторий
```
git clone https://github.com/dimabresky/travelsoft.reviews.git
```
